변수(variable) : 데이터를 저장할 때 쓰이는 '이름이 붙은 저장소'
변수 선언 키워드 : 변수를 선언하고 초기화하는 데 사용되는 특정 키워드
매개변수(parameter) : 함수의 정의에서 전달받은 인수를 함수 내부로 전달하기 위해 사용하는 변수
인수 || 인자(argument): 함수가 호출될 때 값을 전달해주는 값
자료형(DataType) : 변수애 데이터가 저장될 때 어떤 타입의 데이터를 저장할 수 있는지 표현한 것
객체(Object) : 키(key)와 값(value)로 구성된 프로퍼티(property)들의 집합
프로포티(property) : 객체 내부의 속성을 의미 Key는 속성명 value는 속성값
메소드(method) : 객체 내부에 프로퍼티로 정의된 함수
호이스팅 : 인터프리터 코드가 실행하기 전 함수,변수,클래스 또는 임포트의 선언문을 해당 범위의 맨 위로 끌어올리는 것
인터프리터 언어: 순차적으로 실행되고 즉시 결과값을 반환하는 프로그램밍 언어
