터미널 열기 : ctrl + ` 
한줄 주석 : ctrl + / 
여러줄 주석 : alt+shift+a
파일 검색 : ctrl + p
자동 정렬 : ctrl + k + f
코드를 함수나 변수로 감싸주는 기능 : ctrl + shift + r
한줄 복사 : shift + alt + 화살표 위 || 아래
코드 한줄 이동 : alt + 화살표 위 || 아래 
한줄 선택 : ctrl + l
같은 단어 하나씩 선택 : ctrl + d
같은 단어 전체 선택 : ctrl + shift + l
멀티 커서 : alt + 클릭
길다란 커서 : ctrl + alt + 화살표 위 || 아래
문자를 자동으로 괄호 묶어주기 : 드래그 + (
에러 발생 시 없는 모듈 자동으로 찾기 (import)  : ctrl + .
